# Бэкенд Паприки

## Запуск для разработки

Все команды надо выполнять в корне репозитория.

Скопируйте шаблон файла с переменными окружения:

```bash
cp backend/.env.template backend/.env.dev
```

Заполните `backend/.env.dev` своими переменными.

Запустите все контейнеры в режиме разработки:

```bash
just dev
```

Одновременно с этим в другом терминале выполните:

```bash
just migrate
just superuser
```

Эти команды выполнят миграции и запросят данные для создания суперпользователя.
